First Box

This is the first box, it can be sorted with other boxes in CSS with Order:(1,2,3). Likewise, we can use padding to give white space around each box. You can also use max-width of the parent container if the text inside of this box needs to wrap to a specific size such as 1500px in this example.

Second Box

This is the second box, We can use a class name in css such as .box:nth-child(#){background-color} to set a specific background color for that number child with the class name boxes, this just allows us to avoid using more specific id's and makes the html cleaner

Third Box

This is the third box, we can use flex-basis to display the width of each box (30%) in the parent container. Likewise, we can use box-sizing border-box to deal with the sizes of elements as this value will consider paddings and borders to shrink content boxes
